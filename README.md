# WebGIS_SpatialQuery

## 介绍
WebGIS 空间查询，GeoServer、Openlayers、PostGIS、ol-layerswitcher

## 软件架构
GeoServer
Openlayers
PostGIS
ol-layerswitcher

## 安装环境

1.  安装JDK
2.  安装Tomcat
3.  安装PostgresSQL + PostGIS
4.  安装Geoserver

## 使用说明

1.  修改geoserver的链接，即修改html文件geoserverURL
